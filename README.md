# WhatsApp Bot

A whatsapp bot based on baileys

## Installation
Rename `config.json.example` to `config.json`
## For Windows
Install ffmpeg & curl first 
and remove fbdl & igdl cause its just work on linux
### Cloning this repo

```bash
  git clone https://github.com/Topandhi/botwa
  cd botwa
  npm install
```
### Start bot

```bash
  npm start
```
